Basic conceptions

1. LiveBoard: The webpage where you can watch your code

2. User: People who visit LiveBoard and watch pictrue

3. Scanner: People who uses Sanner App scans QR code / Bar code and share it on LiveBoard

User

1. Visit LiveBoard and input room number(eg. A1080 / Tom / 000)

2. Then you will come into the room and waiting somebody updates code

3. Remember to close the webpage after using

Scanner

1. You can download the Scanner App(Android platform only) at home page

2. Open the app and scan the QR code / Bar code you want to share

3. Kill the background process after using it.
